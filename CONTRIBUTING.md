## Adding a Project | 添加项目

### What We Accept | 我们接受什么

- Open-source projects related to AI agent development
- Commercial tools with significant community adoption
- Learning resources (courses, books, tutorials) of high quality
- Job boards and career resources relevant to AI agent roles

### What We Don't Accept | 我们不接受什么

- Projects with fewer than 100 GitHub stars (unless exceptionally relevant)
- Unmaintained projects (no commits in the last 12 months)
- Pure marketing pages without real product/code
- Duplicate entries or minor forks of existing listed projects

### How to Add | 如何添加

1. Fork this repository
2. Find the correct section based on the project's primary use case
3. Add your entry in the appropriate table, maintaining alphabetical order within each section
4. Ensure your entry follows the format below
5. Submit a Pull Request

### Entry Format | 条目格式

For GitHub projects:
```markdown
| [Project Name](https://github.com/owner/repo) | Xk+ | Brief description in English. |
```

For non-GitHub resources:
```markdown
| [Resource Name](https://url.com) | — | Brief description in English. |
```

**Requirements | 要求:**
- Description should be one sentence, max 100 characters
- Use present tense ("Provides...", "Enables...", not "Provided...")
- No promotional language ("The best...", "Amazing...", "Revolutionary...")
- Include both English description (required) and Chinese description (optional but encouraged)

## Quality Standards | 质量标准

Every entry must meet these criteria:

每个条目必须满足以下标准：

| Criterion | Requirement |
|---|---|
| **Relevance** | Directly related to building, deploying, or learning about AI agents |
| **Activity** | Active maintenance (commits within last 12 months) |
| **Quality** | Well-documented, has a clear README |
| **Popularity** | 100+ GitHub stars OR significant community adoption |
| **Uniqueness** | Not a trivial fork or duplicate of an existing entry |

## PR Guidelines | PR 规范

### Before Submitting | 提交前

- [ ] Search existing entries to avoid duplicates
- [ ] Verify all links are working
- [ ] Follow the existing formatting and style
- [ ] Place the entry in the correct section
- [ ] One PR per addition (unless related entries)

### PR Title Format | PR 标题格式

```
Add [Project Name] to [Section Name]
```

Examples:
- `Add Agno to Agent Frameworks`
- `Update LangGraph description`
- `Fix broken link for Mem0`

### PR Description | PR 描述

Please include:
- What the project does (1-2 sentences)
- Why it belongs in this list
- Which section you're adding it to

## Content Guidelines | 内容规范

### Bilingual Content | 双语内容

This is a bilingual (English + Chinese) project. When adding or editing:

- **English is required** for all entries
- **Chinese translations are encouraged** but not required
- If you can only write in one language, that's fine — others can help translate
- Keep translations natural, not word-for-word literal

### Star Counts | Star 数量

- Use approximate ranges: `1k+`, `5k+`, `10k+`, `25k+`, `50k+`, `100k+`
- Don't use exact numbers (they change daily)
- Use `—` for non-GitHub or private projects

### Section Placement | 板块归属

If a project fits multiple sections, place it in the **primary use case** section and add a brief mention or cross-reference in secondary sections.

### Ordering | 排序

Within each table section, entries are ordered by:
1. Relevance and significance to the category
2. Community adoption (stars/usage)
3. Alphabetical for similar-tier projects

## Reporting Issues | 反馈问题

Found a problem? Open an issue for:

- **Broken links** — Include the entry name and broken URL
- **Outdated information** — Include the current vs correct information
- **Missing projects** — Suggest additions you'd like to see
- **Category improvements** — Suggest better organization

---

## Code of Conduct | 行为准则

- Be respectful and constructive
- Focus on the quality of resources, not self-promotion
- Help maintain the bilingual nature of the project
- Give credit where credit is due

Thank you for helping make this the best resource map for agent builders!

感谢你帮助我们打造 Agent 开发者的最佳资源地图！
